# TaijiOS Lite prototype

> This repository is a lightweight TaijiOS prototype and example pack kept online for historical links.

Current main repository: [yangfei222666-9/taiji](https://github.com/yangfei222666-9/taiji)

Live HUD demo: [taijios-hud.netlify.app](https://taijios-hud.netlify.app)

Reusable ideas should live in the main repo as examples. New development, issues, and public references should use `taiji`.

This repository is an archive candidate. Do not treat it as the canonical TaijiOS codebase.
