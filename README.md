# TaijiOS Lite v1.3.0

**带自进化的AI认知军师。每个实例 = 一个智能体(Agent)，所有Agent互相学习，越用越聪明。**

> 一个人的认知有盲区，一群人的认知才完整。

---

## 它能做什么

- **像诸葛亮一样一针见血** — 不是客服不是心理咨询师，是军师
- **越用越懂你** — 8个引擎并行，从对话中自动学习你的模式
- **用易经诊断你的状态** — 6维度映射64卦，动态切换军师策略
- **跨对话记住你** — 五维认知地图（位置/本事/钱财/野心/口碑）持续积累
- **Agent互相学习** — 导出你的经验给其他Agent，导入别人的经验为你所用
- **兼容12种AI模型** — DeepSeek/GPT/Claude/通义/GLM/豆包/Kimi...随意切换

---

## 八大引擎

```
用户对话
  ├─ 1. 经验结晶引擎 ──── 从对话模式中提取规则，注入system prompt
  ├─ 2. 对话学习引擎 ──── 用下一句话推断上一轮质量，持续反馈
  ├─ 3. 易经卦象引擎 ──── 6维状态 → 64卦 → 16种军师策略
  ├─ 4. AGI认知地图 ───── 跨对话构建五维认知，发现盲区
  ├─ 5. 共享经验池 ────── Agent间经验流通（v2：含易经+灵魂数据）
  ├─ 6. Premium付费层 ─── 免费版够用，付费解锁深度分析
  ├─ 7. 贡献积分系统 ──── 8种积分来源，5个等级
  └─ 8. 生态制度 ────────  Agent网络 + 16成就 + 5角色
```

## 快速开始

### 方式一：下载exe（不需要Python）

从 [Releases](../../releases) 下载 `TaijiOS-Lite.zip`，解压后：

1. 把你的ICI文件（.docx）放进同一个文件夹（没有也行，30秒快速建档）
2. 双击 `TaijiOS-Lite.exe`
3. 首次运行选择AI模型，按步骤获取API Key

### 方式二：从源码运行

```bash
pip install python-docx openai python-dotenv
python taijios.py
```

## Agent互学机制

```
Agent A 对话进化 → 结晶 → 导出 .taiji（v2: 结晶+易经+灵魂）
                                ↓
Agent B 导入 → 规则融入认知 → 验证 → 置信度上升
                                ↓
Agent C 导入A+B → 交叉验证 → 全网经验质量越来越高
                                ↓
新Agent加入 → 直接继承最佳经验 → 起点更高
```

**.taiji v2 经验包** 不只是规则列表，还携带：
- 来源Agent的易经卦象快照（当时的状态）
- 来源Agent的认知模式（匿名化的五维分析）
- 验证计数（被多少Agent采纳过）

## 生态制度

**5个角色：**
| 角色 | 积分 | 说明 |
|------|------|------|
| 探索者 | 0+ | 刚加入，正在了解自己 |
| 贡献者 | 50+ | 开始产出经验 |
| 传播者 | 200+ | 有足够经验帮别人 |
| 导师 | 500+ | 经验被多人采纳 |
| 先行者 | 1000+ | 推动网络进化 |

**16个成就：** 初出茅庐、言之有物、促膝长谈、知己知彼、初见端倪、集腋成裘、乐善好施、兼收并蓄、桃李天下、三日不辍、七日精进、月度修行、问道周易、易理初通、广而告之、生态觉醒

**六条铁律：**
1. 经验靠验证 — 被越多Agent采纳，质量越高
2. 分享越多越强 — 帮别人就是帮自己
3. 多样性优先 — 不同Agent交叉验证最有价值
4. 军师无废话 — 只保留真正有用的规则
5. 开放流通 — .taiji文件自由传播
6. 随时迭代 — 每个Agent随时准备接收新经验

## 易经卦象引擎

| 爻位 | 维度 | 阳(积极) | 阴(消极) |
|------|------|----------|----------|
| 初爻 | 情绪基底 | 情绪稳定 | 焦虑挫败 |
| 二爻 | 行动力 | 有目标在行动 | 迷茫纠结 |
| 三爻 | 认知清晰度 | 自我认知清 | 混沌不清 |
| 四爻 | 资源状态 | 有资源支持 | 匮乏缺乏 |
| 五爻 | 方向感 | 方向明确 | 摇摆不定 |
| 上爻 | 满意度 | 正面为主 | 负面为主 |

16种核心军师策略：乾(进攻)、坤(防守)、屯(启动)、蒙(启蒙)、需(等待)、讼(调解)、师(统帅)、比(联盟)、观(点破)、剥(止损)、复(复苏)、困(破局)、渐(稳进)、既济(居安)、未济(冲刺)、涣(聚焦)

## 什么是ICI

ICI = Individual Cognitive Identity = 个体认知身份标识

把一个人拆成5个维度（位置、本事、钱财、野心、口碑），形成完整的认知档案。TaijiOS读取这个档案，激活为你的专属认知军师。

没有ICI文件也能用——7个问题30秒快速建档。
