# TaijiOS Lite

带自进化的ICI认知AI军师。读取你的ICI档案，五引擎并行进化，越用越懂你。

## 五引擎架构

```
用户对话
  ├─ 经验结晶引擎 ──── 从对话模式中提取规则，注入system prompt
  ├─ 对话学习引擎 ──── 用下一句话推断上一轮质量，持续反馈
  ├─ 易经卦象引擎 ──── 6维状态 → 6爻 → 64卦 → 军师策略切换
  ├─ AGI认知地图 ───── 跨对话构建用户五维认知，发现盲区
  └─ 共享经验池 ────── 跨用户经验流通（export/import .taiji文件）
```

## 特性

- **ICI档案解读** — 读取.docx格式的ICI文件，AI激活为你的认知军师
- **快速建档** — 没有ICI文件？7个问题30秒建档
- **经验结晶** — 从对话模式中自动提取规则（避免/保持/发现三类）
- **易经卦象** — 对话状态实时映射到64卦，军师策略动态切换
- **AGI认知地图** — 跨对话积累五维认知（位置/本事/钱财/野心/口碑），发现用户盲区
- **共享经验** — 导出你的经验，导入别人的，验证越多置信度越高
- **收费层** — 免费版基础功能齐全，Premium解锁深度分析
- **军师模式** — 诸葛亮式一针见血，禁止废话

## 快速开始

### 方式一：下载exe（不需要Python）

从 [Releases](../../releases) 下载 `TaijiOS-Lite.zip`，解压后：

1. 把你的ICI文件（.docx）放进同一个文件夹
2. 双击 `TaijiOS-Lite.exe`
3. 首次运行粘贴 DeepSeek API Key（去 [platform.deepseek.com](https://platform.deepseek.com) 注册）

### 方式二：从源码运行

```bash
pip install python-docx openai python-dotenv
python taijios.py
```

## 自进化原理

```
你说话 → AI回复 → 你的下一句话 → 系统推断上一轮好不好
                                        |
                          正面/负面记录到 soul_outcomes.jsonl
                                        |
                          每10轮自动触发结晶引擎
                                        |
           +----------------------------+----------------------------+
           |                            |                            |
     经验结晶注入                   卦象策略切换                 认知地图更新
     system prompt                军师风格动态变             五维盲区检测
```

## 易经卦象引擎

六爻对应：
| 爻位 | 维度 | 阳(积极) | 阴(消极) |
|------|------|----------|----------|
| 初爻 | 情绪基底 | 情绪稳定 | 焦虑挫败 |
| 二爻 | 行动力 | 有目标在行动 | 迷茫纠结 |
| 三爻 | 认知清晰度 | 自我认知清 | 混沌不清 |
| 四爻 | 资源状态 | 有资源支持 | 匮乏缺乏 |
| 五爻 | 方向感 | 方向明确 | 摇摆不定 |
| 上爻 | 满意度 | 正面为主 | 负面为主 |

16种核心军师策略（覆盖最常见状态）：
乾(进攻型)、坤(防守型)、屯(启动型)、蒙(启蒙型)、需(等待型)、讼(调解型)、师(统帅型)、比(联盟型)、观(点破型)、剥(止损型)、复(复苏型)、困(破局型)、渐(稳进型)、既济(居安型)、未济(冲刺型)、涣(聚焦型)

## 什么是ICI

ICI = Individual Cognitive Identity = 个体认知身份标识

把一个人拆成5个维度（位置、本事、钱财、野心、口碑），每个维度3个藏三方，加上认知结构和功能组，形成完整的认知档案。
